# arquitetura-p2p

## comandos no terminal do peer
1)  ``get <filename>``
    - requisita o arquivo para outros peers
2) ``myfiles``
    - retorna a lista de arquivos do peer
3) ``whohas <filename>``
    - retorna lista de endereços de peers que possuem o arquivo
4) ``exit``
    - finaliza conexão do peer
